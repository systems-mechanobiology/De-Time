# Contributing

De-Time follows the repository-level contribution, conduct, and security
documents:

- [`CONTRIBUTING.md`](https://github.com/systems-mechanobiology/De-Time/blob/main/CONTRIBUTING.md)
- [`CODE_OF_CONDUCT.md`](https://github.com/systems-mechanobiology/De-Time/blob/main/CODE_OF_CONDUCT.md)
- [`SECURITY.md`](https://github.com/systems-mechanobiology/De-Time/blob/main/SECURITY.md)

When contributing code, prefer changes that strengthen the canonical `detime`
package first. New benchmark workflows should be proposed against the companion
`de-time-bench` repository rather than against the main package.
