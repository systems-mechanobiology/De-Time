# Quant Trading Tutorial Repository Type

This tutorial area is a mixed repository object:

| Type | Why it applies | Governance used |
|---|---|---|
| Platform/software example | Reusable code modules expose data loading, feature construction, strategy recipes, and backtest comparison. | API-style modules, smoke scripts, documented commands. |
| Benchmark/tutorial repository | Classical strategies are compared with decomposition-aware variants on real market data. | dataset passport, baseline passport, metric passport, experiment matrix, run manifest. |
| Manuscript/tutorial evidence object | Notebook prose makes a methodological claim: decomposition makes trend, cycle, residual, and volume structure explicit. | claim-to-evidence map, strategy-signal map, reproducibility checklist. |

Scope boundary: this is an educational research column, not a live-trading system. Results from bundled sample data or smoke tests must not be used as production trading evidence.
